Budget Menu

Contains all the functionality for Budget App

Tutorial Review

This tutorial can be found here for future reference

"Budget" tab

Setup your budget: This can be changed at any time

Budget Menu Options

Remove and Add Lines (below)

"Income" tab

Setup Income: Menu options are the same as Budget

"Current Month" tab

Budget Layout: Your budget has 4 sections

Expense Tracker

Expense Dropdown: Maps to your up-to-date budget

Income Tracker

Income Dropdown: Maps to your up-to-date income

Bill Reminders

Expense/Bill Connected: Items set as Bill in Budget give additional info

Bill Reminders cont...

Partial Bill Payment: Bills not fully paid show as "Partial"

Tracker Sections

Info: Each Tracker Section (and Bills Section) show totals on the header

Start New Month

You will be prompted to confirm. The current month being stored can still be edited.

Last Month Data

The "Last Month" column and "Last Month Ending Balance" row will give you the balance at the end of last month.

Ran Out of Rows

If you have run out of rows in either the Expense or Income Tracker sections...

Ran Out of Rows cont...

Budget Menu ➔ Month Options ➔ Need More Rows...

Ran Out of Rows cont...

In the Pop-up select which section you need more rows in and how many, then select "Submit"

Ran Out of Rows cont...

After a moment rows will be added to the tracker you chose

Monthly Summary

You can review previous months as a snap-shot from here (at least one month must be store to see this menu).

Monthly Summary cont...

Use the dropdown to select what month you want and you get all the information the left Budget section on the current month gives you.

Edit Previous Months

You have the ability to edit previous months with this menu option.

Edit Previous Months cont...

After making any changes store the month again with this menu option.

Message Notifications (Toasts)

Keep an eye out for important message notifications at the bottom right of the screen.

Resize Columns

When a column needs to be bigger, double click inbetween columns to expand them.

Additional Help (Contacts)

If an error message appears that informs you to contact support or if you have any questions, select this menu options.
